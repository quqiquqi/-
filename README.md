# 南京审计大学校园二手物品交易平台

## 项目简介

南京审计大学校园二手物品交易平台是一个专为南审学子设计的在线交易系统，旨在方便学生之间进行二手物品的买卖，促进资源的循环利用，减少浪费。

## 功能特性

- **用户认证系统**：注册、登录、个人信息管理
- **物品发布功能**：上传物品信息、图片、价格等
- **物品浏览和搜索**：按分类浏览、关键词搜索
- **交易流程管理**：联系卖家、交易状态跟踪
- **响应式设计**：适配电脑、平板和手机等多种设备

## 技术栈

- **前端**：HTML5、CSS3、JavaScript
- **样式框架**：Tailwind CSS v3
- **图标库**：Font Awesome 4.7.0
- **图片服务**：Trae API 文本转图片

## 项目结构

```
├── index.html          # 首页
├── login.html          # 登录页面
├── register.html       # 注册页面
├── post.html           # 发布物品页面
├── profile.html        # 个人中心页面
└── README.md           # 项目说明
```

## 如何使用

1. 打开 `index.html` 进入首页
2. 点击 "注册" 按钮创建新账户
3. 登录后可以发布二手物品或浏览其他用户发布的物品
4. 在个人中心查看交易记录和管理个人信息

## 功能说明

### 首页
- 展示推荐物品列表
- 提供搜索功能
- 分类导航

### 登录/注册
- 支持邮箱注册和登录
- 支持微信、QQ社交登录（模拟）

### 发布物品
- 上传物品图片（最多5张）
- 填写物品标题、分类、价格、描述等信息
- 设置交易地点和联系方式

### 个人中心
- 管理个人信息
- 查看发布的物品
- 查看交易记录
- 管理收藏的物品

## 注意事项

- 本项目为前端展示版本，所有功能均为模拟实现
- 实际使用时需要后端服务支持数据存储和处理
- 图片上传功能仅为前端展示，实际上传需要后端处理
- 社交登录功能需要集成相应的第三方API
